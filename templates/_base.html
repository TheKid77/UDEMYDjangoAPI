Andy's IMDB Style API Django Rest Framework Collection

This API Collection allows you to view all distribution platforms, movies and reviews. Users can also create new movies and create individual reviews for a movie.

Learn More about DRF

User Guide

Registration and Authentication

Click on Swagger API Documentation

Use the /account/register/ endpoint (Try it out) to create a new user. (Requires a Username, Email and Confirmed Password)

Use the /account/login/ endpoint (Try it out) to login which will return an authentication tokenstring. (Requires a Username and Password.)

Click on Authorize and supply the value, "Token tokenstring" - Click Authorize.

Create a new movie

Click on Swagger API Documentation.

Use the POST /watch/create-movie endpoint (Try it out) to create a new movie. (Only requires a Title, Storyline and Platform

1 - Disney Channel
2 - Prime Video
3 - Netflix

Create a review for a specific movie

Click on Swagger API Documentation.

Use the GET /watch/list/ endpoint (Try it out) and identify the Id of the movie you wish to review (listed above "reviews" field)

Use the POST /watch/{id}/review-create endpoint (Try it out) to create a review. (Requires a Rating, Description and the Id of the movie identified above.)

A user can only review each movie once!
